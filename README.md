# Avangard Jiu-Jitsu Academy — AI Agent Planning Repository

This repository contains the filled planning and workflow documents for a clean restart of the Avangard Jiu-Jitsu Academy website.

## Documents
- `REQ.md` — product goals, audience, requirements, acceptance criteria.
- `CONTEXT.md` — technical architecture, stack, hosting, constraints.
- `DESIGN.md` — visual direction and design rules.
- `STATE.md` — current phase, tasks, risks, next actions.
- `TDD.md` — testing and quality strategy.
- `agent.md` — operating rules for Codex/AI agents.

## Important
Codex/AI agents should not start coding until these documents are reviewed and approved.

Recommended implementation stack:
- Next.js
- TypeScript
- Tailwind CSS
- Vercel
- No database initially
- Lightweight structured content/admin workflow
